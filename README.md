## 🎮 Overview
**MathX: Infinity** is a high-quality CrossMath-style puzzle game built specifically for the **Telegram Mini App** ecosystem. It combines brain-teasing arithmetic challenges with a modern, customizable interface and social competition.

---

## ✨ Key Features
*   🧩 **Infinite Puzzles:** Adaptive grid generation from 3x3 (Easy) up to 9x9 (Expert).
*   ⚔️ **Battle Mode:** Compete against a smart Bot in a race to solve equations.
*   📅 **Daily Challenges:** Sync with the world and solve the unique "Puzzle of the Day".
*   🎨 **Custom Themes:** Switch between **Onyx**, **Amethyst**, **Paper**, and **Cyberpunk** skins.
*   📈 **Global Ranking:** Track your XP and Level via the Supabase-powered leaderboard.
*   📱 **Native Experience:** Full Haptic Feedback support for mobile devices.

---

## 🛠 Tech Stack
*   **Engine:** Vanilla JavaScript + HTML5 Canvas (procedural generation).
*   **API:** FastAPI (Python 3.11) + Uvicorn.
*   **Cloud:** Supabase (PostgreSQL) for user profiles and scores.
*   **Platform:** Telegram WebApp API.

---

## 🚀 Installation (Local)
1. Clone the repository:
   ```bash
   git clone https://github.com/nyutik/math-x-tma.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the development server:
   ```bash
   python main.py
   ```
4. Open `http://localhost:8000` in your browser.
